Dicionário HTML

<p>

Tags de texto parágrafo.

<h1> a <h6>

Tags de texto Título 1 a 6 com h1 mais relevante e h6 menos.

<strong>

Tags de texto destacando a importância.

<br>

Tags de texto quebra de linha única.

<div>

Tags de divisão de texto HTML.

<a>

Tags de link HTML, âncora para um link.

<img>

Tags de imagem, insere uma imagem.

<link>

Especifica as relações entre o documento atual e um recurso externo.

<title>

Define o título do documento, mostrado na barra de título de um navegador.

<body>

Representa o conteúdo de um documento HTML.

<video>

Utilizado para incorporar conteúdo de vídeo em um documento HTML.
